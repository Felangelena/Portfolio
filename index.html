Technologies

What I use in my work

HTML5 is the latest version of Hypertext Markup Language, the code that describes web pages. HTML5 can be used to write web applications that still work when you're not connected to the net; to handle high definition video; and to deliver extraordinary graphics.
CSS3 is the latest evolution of the Cascading Style Sheets language. It brings a lot of new features and additions, like rounded corners, shadows, gradients, transitions or animations, as well as new layouts like multi-columns, flexible box or grid layouts.
JavaScript is a scripting language used to create and control dynamic website content, i.e. anything that moves, refreshes, or otherwise changes on your screen without requiring you to manually reload a web page.
jQuery is a fast, small, cross-platform and feature-rich JavaScript library. It is designed to simplify the client-side scripting of HTML. It makes things like HTML document traversal and manipulation, animation.
SCSS is a preprocessor language that is compiled into CSS. Using SCSS add many additional functionalities to CSS such as variables, nesting and more. All these additional functionalities can make writing CSS much easier and faster as compared to writing the traditional CSS.
Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains CSS- and (optionally) JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.
BEM is a highly useful, powerful, and simple naming convention that makes front-end code easier to read and understand, easier to work with, easier to scale, more robust and explicit, and a lot more strict.
Visual Studio Code is a code editor redefined and optimized for building and debugging modern web and cloud applications. Visual Studio dev tools & services make app development easy for any platform & language.
Gulp is a command-line task runner for Node.js. Gulp let us automate processes and run repetitive tasks with ease. These tasks are code-based and use plugins. The tasks modify the files, building source files into production files.
Figma is a web-based graphics editing and user interface design app. It can be used to do all kinds of graphic design work from wireframing websites, designing mobile app interfaces, prototyping designs, crafting social media posts, and everything in between.
MAMP is a popular local web development tool. It turns computer into a server environment that can host websites. MAMP uses Apache, MySQL, and PHP, making it highly compatible with WordPress.
